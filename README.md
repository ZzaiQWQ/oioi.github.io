## ✨ 项目特色 (Highlights)

* **🐾 互动像素猫**：内置一只具有自主行为的像素猫，它会随机巡逻、发呆、坐下或趴下。你可以点击它进行互动，气泡对话框会随它的动作动态位移。
* **🖼️ 多相册管理系统**：支持通过 `photos.js` 配置多个本地或在线相册，具有优雅的相册封面展示和照片灯箱预览功能。
* **🎨 双色主题切换**：支持 **Zzai 粉**（治愈）与 **Sky 蓝**（静谧）两种主题模式，鼠标移动时会产生与主题颜色匹配的星星尾迹。
* **📝 治愈系任务清单**：每日自动从任务池中随机生成三条“小确幸”任务（如：陪猫发呆、给猫投喂），带你感受慢生活。
* **💻 拟真 OS 体验**：拥有响应式的任务栏（Taskbar）、窗口管理（支持窗口隐藏/恢复）、桌面图标以及实时更新的系统时间。

## 🛠️ 快速开始 (Getting Started)

本项目是一个纯前端项目，无需配置复杂的开发环境，即可本地运行。

1.  **克隆项目**
    ```bash
    git clone [https://github.com/你的用户名/Zzai-OS.git](https://github.com/你的用户名/Zzai-OS.git)
    ```
2.  **配置照片**
    * 将你的照片放入 `images/` 文件夹。
    * 在 `photos.js` 文件中按照格式添加你的相册分类和文件路径。
3.  **直接运行**
    * 双击 `index.html` 或使用 VS Code 的 `Live Server` 插件运行即可。

## ⚙️ 技术细节 (Tech Stack)

* **Core**: [React 18](https://reactjs.org/) (通过 Babel 动态解析)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/)
* **Typography**: M PLUS Rounded 1c, VT323 (像素风), Indie Flower (手写风)
* **Animations**: 原生 CSS 动画驱动的像素猫行为逻辑
